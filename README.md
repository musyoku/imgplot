# imgplot

imgplot is a non-blocking image viewer for Python 3.

# Requirements

- C++14 (gcc-6)
- OpenGL 4
- Python 3

# Installation

**GLFW**

```
sudo apt install libglfw3-dev
```
**pybind11**

```
pip3 install pybind11 --user
```
**imgplot**

```
make
```
